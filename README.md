# SkillSwap-AI
SkillSwap AI — Платформа для P2P обміну навичками
Пітч: SkillSwap AI — це інтелектуальна P2P-платформа для обміну знаннями за принципом бартеру. ШІ аналізує рівень навичок та графік, щоб підібрати ідеального партнера для безоплатного навчання.
Проблема: люди часто не мають коштів на дорогі курси, хоча самі володіють цінними навичками, якими могли б поділитися.
Мета першої ітерації: Запустити закритий бета-тест для перших 100 користувачів.  
MVP: Можливість користувача зареєструвати свій скіл-сет, отримати один збіг (match) від ШІ та провести відеоурок.
Definition of Success: Це конкретні вимірювані тести.  
Здатність виконати головний сценарій (реєстрація + матч) без зайвих кроків.  
Стабільність відеозв'язку протягом годинної сесії.  
Здатність нової людини знайти вчителя без підказок.
Карта стейкхолдерів:
Користувачі: Студенти, фрілансери та люди, що змінюють професію.  
Власник: Оцінює досягнення цілей та визначає межі ітерації.  
Розробники: Відповідають за інтеграцію АІ-моделей та стабільність серверів.
