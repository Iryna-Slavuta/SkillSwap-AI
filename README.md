# SkillSwap-AI
SkillSwap AI — Платформа для P2P обміну навичками
## Пітч:
SkillSwap AI — це інтелектуальна P2P-платформа для обміну знаннями за принципом бартеру. ШІ аналізує рівень навичок та графік, щоб підібрати ідеального партнера для безоплатного навчання.

## MVP:
Можливість користувача зареєструвати свій скіл-сет, отримати один збіг (match) від ШІ та провести відеоурок.

## Мета першої ітерації:
Запустити закритий бета-тест для перших **100 користувачів**, де принаймні 40% успішно знайдуть партнера й проведуть перший урок протягом 2 тижнів

## Карта стейкхолдерів

| Стейкхолдер | Очікування | Вплив |
|-----------|-----------|-------|
| **Учні (студенти, фрілансери)** | Безоплатне навчання, зручний матчинг, гнучкий графік | Найвищий — основні користувачі |
| **Вчителі** | Визнання своїх скіл-сетів, отримання часових токенів | Найвищий — якість навчання залежить від них |
| **ШІ-система** | Точність матчингу, швидкість обробки запитів | Критичний — без цього платформа неконкурентна |
| **Адміністратори платформи** | Стабільність, безпека даних, моніторинг якості | Середній — роблять кроки поліпшення |
| **Інвестори/спонсори** | Доказ життєздатності моделі, зростання користувачів | Середній — визначають фінансування |

## Definition of Success:
Здатність виконати головний сценарій (реєстрація + матч) без зайвих кроків.  
Стабільність відеозв'язку протягом годинної сесії.  
Здатність нової людини знайти вчителя без підказок.
